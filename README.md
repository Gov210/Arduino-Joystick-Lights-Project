# Arduino-Joystick-Lights-Project

This is a simple Arduino project that uses a joystick module to control LEDs. This project demonstrates how analog joystick inputs can be used to control digital outputs and is a great introduction to Arduino programming.

Features:
- Control LEDs using a joystick
- Read analog X and Y axis values
- Demonstrates analog input and digital output
- Beginner-friendly Arduino project

Components:
- Arduino Uno (or compatible board)
- Joystick module
- LEDs
- 220Ω resistors
- Breadboard
- Jumper wires
